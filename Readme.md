# HTML & CSS Selectors practice project
## 📌Project overview 
This project demonstrate the use of ** HTML and CSS Selectors** to style web pages without using javascript or external libraries.It is created for learning and practicing how different selectors work in real web development.
## 🎯 Features
-uses only Html and CSS
-Demonstrate different CSS selectors:
-Element selector
-Class selector
-ID selector
-Group selector
-universal selector
-Descendent selector
-child selector
## How to run
->open folder that Task 3
-> right click on index.html file
->open in browser
